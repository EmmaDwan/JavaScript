1. html是 标记语言: 被读取，不用于向计算机发指令 常用于格式化和链接。 被动的 JavaScript是 编程语言: 很强逻辑性和行为能力的指令。 主动的 2.计算机语言分为三大类： 机器语言 计算机执行的 汇编语言 高级语言 C C++ java C# python PHP JavaScript Go Objective-C Swift 后两者叫编程语言 前两者都是直接对硬件操作 3.JavaScript是运行在客户端的脚本语言,脚本语言就是不需要编译， 运行过程中由js解释器(js引擎) 逐行解释并执行 可以基于node.js技术进行服务器端编程。既可以做前端，也可以做后台 4.JavaScript作用: 表单动态检验(e.g.密码是否合规格的检测) (最初目的) 网页特效 服务端开发(node.js) 桌面程序(electron) APP(Cordova) 控制硬件——物联网(Ruff) 游戏开发(cocos2d-js) html 身体 css 穿衣化妆 JavaScript 做动作 5.浏览器执行js 浏览器分为两部分： 渲染引擎和js引擎 (1)渲染引擎:解析html、css，内核 比如Chrome的blink，老版本的webkit (2)js引擎: JS解释器逐行读取——转化机器语言——运行js代码 比如chrome的V8(号称最快) 6.JS的组成 ECMAScript 规定了js编程语法和核心知识，所有浏览器厂商遵循的工业标准 DOM 页面文档对象模型 一种标准编程接口 通过DOM提供的接口对页面元素进行操作 BOM 浏览器对象模型 独立于内容，可以和浏览器窗口进行互动的对象结构 通过BOM可以操作浏览器窗口，e.g.弹出框、跳转获取分辨率
