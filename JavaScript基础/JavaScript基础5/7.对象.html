(1)对象：是一个具体的事物 e.g. 苹果 不是 游戏 不是 这个苹果 是 绝地求生 是 js中对象：一组无序的 相关属性和方法的集合，属于复杂数据类型object 1.所有的事物比如字符串、数值、数组，都可以是对象 2.js对象令表达结构更清晰、更强大 (2)对象的组成 属性：事物的特征 (常用名词) ———— 和 变量 区分开 方法：事物的行为 (常用动词) ———— 和 函数 区分开 (3)创建对象： 1.利用对象字面量 {} 1)属性名 配 属性值(键值对) 2)逗号,隔开 3)方法后面跟的是 匿名函数 var 对象名 = { 属性1: 值1, 属性2: 值2, 属性3: 值3, 属性4: 值4, 方法1:function(形参) { } } 2.利用new Object var 对象名 = new Object(); 对象名.属性1 = 值1； 对象名.属性2 = 值2； 对象名.属性3 = 值3； 对象名.属性4 = 值4； 对象名.方法 = function(形参) { } 3.利用构造函数 （前两种一次只能创建一个对象，创建很多对象是大量重复的） 把对象里一些 相同的属性和方法 抽象出来封装到函数里。 这个函数不一样，里面封装不是普通代码，而是对象 (1)构造： function 构造函数名(形参1，形参2，形参4，形参4) { this.属性1 = 形参1; this.属性2 = 形参2; this.属性3 = 形参3; this.属性4 = 形参4; this.方法名 = function(形参5) { } } (2)调用： var 对象名 = new 构造函数名(); (调用构造函数，返回的是一个对象) (4)调用对象 1 对象名.属性名 gabrielle.uname 对象名['属性名'] gabrielle['uname'] 2 对象名.方法() gabrielle.sayHi()
