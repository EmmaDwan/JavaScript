(1) 面向过程的思想(POP)： process-oriented programming 分析出解决问题需要的步骤，把这些步骤一步一步实现，到时候一个一个依次调用 (按步骤划分————蛋炒饭，混合在一起) e.g. 1.打开冰箱门 2.大象放进去 3.关上冰箱门 优：性能高，适合和硬件练习紧密的，比如单片机 缺：不易维护、复用、扩展 (2) 面向对象的思想(OOP)： object oriented programming 把事物分解成一个一个对象，由对象之间分工合作：封装性 继承性 多态性 (按对象划分————盖浇饭) e.g. 1.大象：进去 2.冰箱：打开 关闭 优：易维护、复用、扩展，可以设计出低耦合的系统 缺：性能低 (3)ES6中的类和对象 1.类：面向对象的思维特点：抽取对象 共同属性和行为————组织成一个类 2.对象：将 类 实例化，获取对象 (不断创建对象，指挥对象做事情) "类"抽象了对象的公共部分 ———— "对象"特指具体某一个，通过类实例化 ES6中新增了 类 的概念，可以用class声明一个类
