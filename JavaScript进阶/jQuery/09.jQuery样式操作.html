jQuery样式操作: 可以用css方法 修改简单元素样式、操作类、修改多个样式 (1).修改元素样式 $(this).css("color"); 参数 只写属性名， 返回属性值 $(this).css("width","300px"); 参数 写属性名、属性值， 修改属性值 $(this).css( {"color":"pink", "font-size":"20px"} ) 参数为 对象形式，设置多组样式 (2).操作类 $("div").addClass(" 类名 ") 这列类名不加. 添加类名 $("div").removeClass( ) 移除 $("div").toggleClass( ) 切换 1 原生JS中: className会 覆盖原先的类名 2 jQ中: 类操作只对指定类名操作(追加类名)，不影响原类名
